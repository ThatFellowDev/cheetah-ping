Monitoring Data

Change History

Full page snapshots, word-level diffs, AI summaries. Auto-pruned by plan tier (7-180 days). Immutable audit trail of all detected changes.

Auto-Pruning · Plan-Tiered Retention

Design Principles

Serverless First No servers to manage. Vercel + Workers + Neon scale to zero. Near-Zero Cost Workers monitoring engine costs $0.04/mo per user at scale. Graceful Degradation Every external service is optional except the database. AI-Enhanced Smart URL analysis + plain-English change summaries via Groq. Plan-Gated Tier limits enforced at both API and Worker level. Privacy-First No tracking pixels. Minimal data collection. Auto-pruned history.

Data stored per layer

  • session token, expiresAt, userId

  • monitors url, label, selector, keyword, checkIntervalMinutes, lastSnapshot, status, consecutiveErrors

  • Auto-pause threshold 3 consecutive errors
  • Snapshot max size 100KB
